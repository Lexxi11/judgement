KURZANLEITUNG

So füllst du das Dokument aus

  1. Dokumentkopf: Titel und Name eintragen und das Fach aus der Liste auswählen. Das heutige Datum wird automatisch bei „Stand“ gesetzt.
  2. Struktur: Mit einer Hauptüberschrift beginnen und darunter Unterüberschriften, Texte oder Definitionen ergänzen.
  3. Inhalt: Für jeden Inhalt den passenden Block wählen. So bleiben Vorschau, Word und Markdown sauber formatiert.
  4. Speichern: Das Projekt als JSON sichern oder direkt nach Word, Markdown, JPG, HTML beziehungsweise PDF exportieren.
Tipp: Vorhandene Mitschriften kannst du über „Word importieren“ oder „Text automatisch importieren“ strukturiert übernehmen.
